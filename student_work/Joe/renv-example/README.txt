Reproducible script instructions

-Unzip this folder and save all files to a local folder
-Save your dataset to the subfolder 'Data' within your new local folder
-Open 'renv-example.Rproj' which should launch Rstudio. Now open "thinkCausal_script.R"
-Install the package `renv` if you do not already have it: run `install.packages('renv')` within the "Console" pane
-Run `renv::restore()` within the "Console" pane
-Execute each line individually or click 'Run' in the top right of the Rstudio script editor
